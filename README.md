## Run Locally

Clone the project

```bash
  git clone https://github.com/e-ramos18/dss-final-project.git
```

Go to the project directory

For Frontend

```bash
  cd dss-final-project
  cd frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```

For Backend

```bash
  cd dss-final-project
  cd backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
